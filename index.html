PIF Model: An AI model to predict and generate video frames

An Artificial Intelligence model that can infer a new frame between two existing sequential frames of a video. We trained a GAN neural network framework using youtube footage to predict its existing frames. The model was able to infer new frames from high definition videos. Furthermore, the model was able to generate new videos from the input data sample. The model works by splitting a video into contiguous frames categorized as scenes, these scenes are processed and passed into the generator network which generates new video frames.

SQL Query Evaluator: An SQL engine to evaluate sql queries

An SQL query evaluator for a single-threaded database that can run TPC-H queries. It has operational support for Select, Project, Join, Union, Aggregate operations, and implements standard optimization techniques such as projection pushdown, selection pushdown, and cross-product to join conversion. Our program will receive an SQL File with a collection of CREATE TABLE and SELECT statements. These statements define a representational schema for the data and perform filtering and other operations on the data. The result of these operations will be in a standardized row-oriented data form.
