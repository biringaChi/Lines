SQL Query Evaluator: An SQL engine to evaluate sql queries

An SQL query evaluator for a single-threaded database that can run TPC-H queries. It has operational support for Select, Project, Join, Union, Aggregate operations and implements standard optimization techniques such as projection pushdown, selection pushdown and cross product to join conversion. Our program will receive an SQL File with a collection of CREATE TABLE and SELECT statements. These statements define a representational schema for the data and performs filtering and other operations on the data. The result of these operations will be in a standardized row-oriented data form.
